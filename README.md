# FINANCIAL SYSTEM WITH JAVA

Java-based academic project developed to demonstrate object-oriented programming concepts, data management, and application structure using Java.

## 📚 About the Project

This repository contains the source code for **Trabalho4Java**, a university assignment/project created in Java.
The project focuses on applying core software development concepts such as:

* Object-Oriented Programming (OOP)
* Class organization
* Data manipulation
* Java application structure
* Clean code practices

## 🚀 Technologies Used

* [Java](https://www.java.com/?utm_source=chatgpt.com)

## ⚙️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/KAE4U/Trabalho4Java.git
```

2. Open the project in your preferred Java IDE:

* IntelliJ IDEA
* Eclipse
* NetBeans

3. Compile and run the `Main` class.
